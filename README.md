# Not Adventure Capitalist

A Python-based idle clicker game built with **Pygame**. Manage resources, unlock new tiers, and automate your production to become a virtual multi-billionaire.

## How to Play

1. **Manual Start:** Click the colored circles to begin a task. When the progress bar finishes, you earn value.
2. **Progression:** - **Green** earns you Money.
   - **Red** increases the value of Green.
   - **Orange** increases the value of Red... and so on!
3. **Upgrades:**
   - **Buy More:** Increases the value earned per cycle.
   - **Managers:** Automates the task so you don't have to click manually.
   - **Boosters:** Increases the speed of the progress bar.
4. **Unlocks:** Reach specific value milestones in one color to unlock the next tier.

## Requirements

- **Python 3.x**
- **Pygame** library
